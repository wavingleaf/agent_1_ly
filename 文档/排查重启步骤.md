# 重启服务并清除 localStorage 再测试

### 步骤

1. 终端 `Ctrl+C` 停掉当前服务

2. 重新启动：
   - **推荐**：在 VSCode 中右键 [启动服务.py](../启动服务.py) → "在终端中运行 Python 文件"
   - **或手动**：
     ```bash
     cd d:/Github项目/agent项目/agent_langchain_1_ly
     D:/Apps/Python/python.exe -m uvicorn src.main:app --host 0.0.0.0 --port 8000
     ```
   - ⚠️ **不要用 `--reload`**：Windows 上 uvicorn + watchfiles 有子进程 spawn 失败 bug（见踩坑记录 #15），文件变更后关闭旧进程但不启动新进程

3. 观察终端启动日志——应该看到两行：
   ```
   AsyncSqliteSaver 初始化完成，对话历史将持久化到磁盘
   ```
   如果没有这行，说明 lifespan 没执行，直接报告。

4. 浏览器打开 `http://localhost:8000`，**按 `F12` 打开 DevTools → Application → Local Storage → 删除 `agent_ui_mode`**（确保从干净状态开始）

5. 下拉框选择 **DST 助手**

6. 在 DevTools → Network 标签页保持打开，然后输入测试消息：
   ```
   帮我查一下 DST 源码中 SetDomestication 函数在哪里定义的
   ```

7. 点击发送后，在 Network 标签页找到 `chat/stream` 请求，点击查看 **Payload** 标签——确认 `mode` 字段值为 `"dst"`

8. 同时观察终端日志——应该出现类似：
   ```
   agent_node mode=dst tools=['calculator', 'grep', 'read_file', 'list_files', 'web_search']
   ```
   如果没出现这行，报告完整启动日志。
